# HTML-Only Multi-Page Website (ElectroHub)

This project is a **pure HTML website** created for the assignment:

- ✅ No CSS
- ✅ No JavaScript
- ✅ Multi-page structure
- ✅ Uses many core HTML elements (semantic tags, tables, forms, media, iframe, etc.)

## Pages

- `index.html` (Home)
- `about.html`
- `services.html`
- `contact.html`
- `form.html`
- `media.html`

## How to Run

1. Download or clone the repository
2. Open `index.html` in your browser

## Folder Structure
